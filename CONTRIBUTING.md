# Contributing

Thank you for contributing to Fetch!

## Ways to Help

- Improve documentation
- Suggest protocol changes
- Report bugs
- Build clients
- Review pull requests

## Pull Request Process

1. Fork the repository.
2. Create a feature branch.
3. Commit your work.
4. Open a pull request.

## Commit Examples

- docs: update specification
- feat: add printer capability
- fix: clarify discovery
