# Code of Conduct

## Our Pledge

We aim to build a welcoming, respectful, and collaborative community.

## Expected Behavior

- Be respectful
- Be constructive
- Welcome newcomers
- Discuss ideas, not people

## Unacceptable Behavior

- Harassment
- Hate speech
- Personal attacks
- Spam
- Deliberate disruption

## Enforcement

Maintainers may remove content or contributions that violate this Code of Conduct.
